# vue3-invoice-app

> CRUD invoice application using Vue 3, Vuex, Vue router & Firebase.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Deploying app to github pages
```
npm run deploy
```
